# insulinska-pumpa-upravljanje-sau

Projekat obuhvata modelovanje i analizu minimalnog modela glukoza-insulin sistema i dizajn PID regulatora za upravljanje doziranjem insulina.

Obuhvaćeno:
- Nelinearni model i linearizacija oko radne tačke
- Izvod funkcije prenosa
- Analiza stabilnosti (polovi i Nyquist)
- Dizajn PID regulatora

Tema je obrađena iz perspektive teorije upravljanja.
